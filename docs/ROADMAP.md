# Roadmap
## Introduction
The erp-wright project aims to deliver an AI-driven platform for automating legacy ERP systems in small manufacturing environments. This roadmap outlines the key milestones and themes for the project, focusing on the Minimum Viable Product (MVP) and subsequent v1 and v2 phases.

## MVP (Must-Have for Launch)
The following features and components are critical for the MVP launch:
### 1. AI Workflow Agents
* Develop and train AI models for automating repetitive tasks (e.g., PO creation, inventory reconciliation)
* Integrate with legacy ERP systems via secure adapters
### 2. Universal ERP Connectors
* Design and implement pre-built adapters for SAP, Oracle, Infor, and custom databases
* Ensure schema auto-mapping for seamless integration
### 3. Visual Workflow Builder
* Create a drag-and-drop interface for designing, testing, and deploying automations
* Enable factory floor managers to build and deploy process automations without writing code
### 4. Real-Time Ops Dashboard
* Develop a monitoring system for automated processes, exceptions, and ROI metrics
* Provide real-time insights for optimized operations
### 5. On-Prem & Cloud Hybrid Deployment
* Design a hybrid deployment model for the core AI engine (in-cloud) and connectors (on-prem)
* Ensure data compliance and security

## v1 Phase (Post-MVP)
The v1 phase will focus on the following themes and features:
### Theme: Scalability and Performance
* Optimize the AI engine and workflow builder for large-scale deployments
* Implement load balancing and autoscaling for high-traffic environments
### Theme: User Experience and Adoption
* Develop a user-friendly onboarding process for factory floor managers
* Create a knowledge base and support resources for users
### Features:
* **Low-Code Workflow Customization**: Enhance the visual workflow builder with custom node development and integration with external services
* **Legacy System Support**: Expand support for additional legacy ERP systems and custom COBOL systems
* **AI Model Training and Validation**: Develop a framework for continuous AI model training and validation using customer data

## v2 Phase (Future Development)
The v2 phase will focus on the following themes and features:
### Theme: Advanced Automation and Intelligence
* Develop advanced AI models for predictive analytics and process optimization
* Integrate with emerging technologies (e.g., IoT, blockchain) for enhanced automation and security
### Theme: Enterprise-Grade Features and Support
* Implement enterprise-grade security, compliance, and governance features
* Develop a comprehensive support and success program for large enterprise customers
### Features:
* **AI-Driven Process Optimization**: Develop AI models for optimizing manufacturing processes and workflows
* **Integration with Emerging Technologies**: Integrate with IoT devices, blockchain networks, and other emerging technologies for enhanced automation and security
* **Enterprise-Grade Deployment and Support**: Develop a comprehensive deployment and support program for large enterprise customers

## Conclusion
The erp-wright project roadmap outlines a clear path for delivering an AI-driven platform for automating legacy ERP systems in small manufacturing environments. By focusing on the MVP, v1, and v2 phases, we can ensure a successful launch, drive adoption, and continue to innovate and improve the platform over time.
