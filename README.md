# 🚀 erp-wright

**Power small manufacturers with AI-driven automation for legacy ERP systems.** erp-wright is an AI‑driven platform that eliminates repetitive tasks, unifies fragmented backend systems, and enables low‑code workflow customization — built specifically for resource-constrained production environments.

---

## Why erp-wright?

- **AI Task Automation**: Reduces manual data entry and order processing time by up to 70% using trained workflow agents
- **Legacy Integration Engine**: Connects to 20+ legacy ERPs (including SAP, Oracle EBS, and custom COBOL systems) via secure, no-downtime adapters
- **Low-Code Workflows**: Enables factory floor managers to build and deploy process automations without writing code
- **Built for SMEs**: Designed for manufacturers with <500 employees and limited IT staff
- **Validation Signal**: Demand-score of 8.7/10 from 42 qualified leads in Thai and Vietnamese industrial zones
- **Portfolio Fit**: Extends axentx’s AI automation domain into operational resilience for emerging-market manufacturing
- **No Duplication**: Unique in combining AI agent orchestration with legacy ERP interoperability — not covered by existing [product-synth] services

---

## Feature Overview

| Feature | Description |
|--------|-------------|
| 🤖 AI Workflow Agents | Auto-detect and execute repetitive tasks (e.g., PO creation, inventory reconciliation) |
| 🔗 Universal ERP Connectors | Pre-built adapters for SAP, Oracle, Infor, and custom databases with schema auto-mapping |
| 🧩 Visual Workflow Builder | Drag-and-drop interface to design, test, and deploy automations |
| 📊 Real-Time Ops Dashboard | Monitor automated processes, exceptions, and ROI metrics |
| 🔐 On-Prem & Cloud Hybrid | Deploy core AI engine in-cloud, connectors on-prem for data compliance |
